How to use

  1. Download the converted file ".sif"
  2. Get an USB stick, refer to the notes below to choose the correct flash drive
  3. Copy the “screensaver.sif” file in the USB stick, in the root, not inside a folder. There must be only one “.sif” file in this place an no software update file (".suf"). The USB stick can contain any other types of files/folder and even others .suf file, but they have to be placed inside a folder not in the main path
  4. Insert the USB stick on the machine
  5. With the machine turned on, navigate to the "Information screen" and press "Update"
  6. The machine will start the process and will show eventual error messages. Do not disconnect the USB while leds near the port are on!
  7. After the "success" message, you can enjoy your custom screensaver!

USB Drive specifications

The USB supports FAT and exFAT file systems but is limited to the MBR partition standard. As a result, some "large" USB sticks (e.g., 128GB, 256GB) may not work if they use the GPT partition standard. Large USB mass storage requiring more than 150mA may not work properly.
